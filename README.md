# ClickBI

Визуальный конструктор SQL-запросов для анализа CSV и Excel файлов прямо в браузере.

## Возможности

- Загрузка CSV / Excel файлов (один или два для JOIN)
- Визуальный построитель запросов: фильтры, группировка, агрегация, сортировка
- Автогенерация SQL
- Интерактивные графики (Bar, Line, Pie, Scatter)
- Скачивание графика в PNG
- Светлая / тёмная тема

## Стек

- Vanilla HTML / CSS / JavaScript
- [Chart.js](https://www.chartjs.org/)
- [PapaParse](https://www.papaparse.com/)

## Запуск

Открой `index.html` в браузере или используй [GitHub Pages](https://pages.github.com/).
