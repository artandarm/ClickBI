# ClickBI

Визуальный конструктор SQL-запросов для анализа и фильтрации CSV файлов прямо в браузере.

## Возможности

- Загрузка CSV файлов
- Детект пропущенных значений с предложением заполнить средним / медианой / модой / нулем
- Визуальный построитель запросов: фильтры, группировка, агрегация, сортировка
- Три типа JOIN (возможность загрузить второй CSV)
- Автогенерация SQL
- Интерактивные графики (Bar, Line, Pie, Scatter)
- Скачивание графика в PNG
- Экспорт отсортированного CSV 
- Светлая / тёмная тема

## Стек

- Vanilla HTML / CSS / JavaScript
- [Chart.js](https://www.chartjs.org/)
- [PapaParse](https://www.papaparse.com/)
